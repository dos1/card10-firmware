Cordio Stack and Cordio Profiles Documentation

Overview

Cordio Stack and Cordio Profiles implements a BLE Host Stack as defined in the Bluetooth core specification [1] and GATT Profiles, respectively. This documentation is a superset of the BLE Host Layer: GAP, SM, GATT, L2CAP and HCI. It covers API, usage examples and internal event handling.
