.. py:module:: bhi160

``bhi160`` - Sensor Fusion
==========================

.. versionadded:: 1.4

Supports the BHI160 sensor on the card10 for accelerometer, gyroscope, magnetometer and orientation.

**Example**:

.. code-block:: python

   import bhi160
   import utime

   bhi = bhi160.BHI160Orientation()

    while True:
        samples = bhi.read()
        if len(samples) == 0:
            utime.sleep(0.25)
            continue
        # print the latest sample
        print(samples[-1])
        utime.sleep(0.25)
